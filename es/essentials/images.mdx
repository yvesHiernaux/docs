## Imagen

### Usando Markdown

La [sintaxis de markdown](https://www.markdownguide.org/basic-syntax/#images) te permite añadir imágenes usando el siguiente código

```md
![title](/path/image.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser menor de 5MB. De lo contrario, recomendamos alojarlo en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Luego puedes usar esa URL e incrustarla.

### Usando Incrustaciones

Para obtener más personalización con las imágenes, también puedes usar [incrustaciones](/writing-content/embed) para añadir imágenes

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Carga otra página HTML dentro del documento. Se usa más comúnmente para incrustar videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
