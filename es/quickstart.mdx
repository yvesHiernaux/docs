## Configura tu desarrollo

Aprende cómo actualizar tus documentos localmente y desplegarlos al público.

### Editar y previsualizar

<AccordionGroup>
  <Accordion icon="github" title="Clona tus documentos localmente">
    Durante el proceso de incorporación, creamos un repositorio en tu Github con
    el contenido de tus documentos. Puedes encontrar este repositorio en nuestro[dashboard](https://dashboard.mintlify.com). Para clonar el repositorio
    localmente, sigue estas[instrucciones](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    en tu terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Vista previa de cambios">
    La vista previa te ayuda a asegurarte de que tus cambios se vean como se pretende. Hemos construido una
    interfaz de línea de comandos para renderizar estos cambios localmente.

    1. Instala el[Mintlify CLI](https://www.npmjs.com/package/mintlify) para previsualizar los
       cambios de la documentación localmente con este comando:`npm i -g mintlify`
    2. Ejecuta el siguiente comando en la raíz de tu documentación (donde está`docs.json`):`mintlify dev`

    <Note>
      Si actualmente estás usando el archivo de configuración heredado`mint.json`, por favor actualiza el Mintlify CLI:

      `npm i -g mintlify@latest`
      Y ejecuta el nuevo comando de actualización en tu repositorio de documentación:

      `mintlify upgrade`
      Ahora deberías estar usando el nuevo archivo de configuración`docs.json`. Siéntete libre de eliminar el archivo`mint.json` de tu repositorio.
    </Note>
  </Accordion>
</AccordionGroup>

### Despliega tus cambios

<AccordionGroup>
  <Accordion icon="message-bot" title="Instala nuestra aplicación de Github">
    Nuestra aplicación de Github despliega automáticamente tus cambios en tu sitio de documentación, así que
    no necesitas gestionar los despliegues tú mismo. Puedes encontrar el enlace para instalar en
    tu[dashboard](https://dashboard.mintlify.com). Una vez que el bot ha sido
    instalado con éxito, debería haber una marca de verificación junto al hash del commit
    del repositorio.
  </Accordion>

  <Accordion icon="rocket" title="Sube tus cambios">
    [Haz commit y sube tus cambios a
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    para que tus cambios se actualicen en tu sitio de documentación. Si subes y no ves que
    la aplicación de Github haya desplegado tus cambios con éxito, también puedes
    actualizar manualmente tu documentación a través de nuestro[dashboard](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
